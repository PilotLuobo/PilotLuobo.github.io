记一次黑苹果安装历程

发表于 2019-08-06 | 分类于 瞎折腾

前言

好久没有折腾黑苹果了，恰好去了一趟扬州老同学家里顺走了一些配件，于是开始琢磨黑苹果的事情了。

硬件准备

硬件 型号 备注
CPU 8700k 顺来的
主板 MSI Z370M迫击炮 顺来的
内存 影驰Gamer 8G * 1 以前用剩下的
硬盘0 三星960evo 256G 顺来的
硬盘1 希捷酷鱼3T 以前用剩下的
电源 海盗船CS450M 嫖来的
机箱 先马 平头哥M1 京东169，送俩风扇
散热 暴雪T400i 顺来的
显示器 戴尔P2416D 打骨折收的

除了机箱其他几乎都是嫖来的，我tql

软件准备

  1. 下载黑果小兵的镜像

    黑果小兵的部落阁

  2. 下载etcher

  3. 寻找合适的efi，或者用通用efi装完再改

  4. 制作镜像U盘

  5. 从优盘启动，开始安装。安装过程基本上是大白话，细节不表。
